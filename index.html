1. Введение

Привет!

Во многих языках программирования функция — это блок кода, который выполняет определённую задачу. Например, получение данных из сети или математические расчёты. Эти «блоки» представляют собой набор инструкций для многократного использования. Обращаться к таким инструкциям можно всякий раз, когда нужно выполнить ту или иную задачу.

Под набором инструкций подразумевается исходный код на языке программирования. В нашем случае это JavaScript.

Этот модуль полностью посвящён функциям в JavaScript. После его изучения вы сможете создавать и использовать функции при решении реальных задач.

Будет интересно! А пока давайте освежим знания по предыдущим темам.

Задание 1.4

Есть код на JavaScript:

let x = 10;

let y = 5;

let z = 2;

if (x > y) {

console.log("x больше y");

} else if (x === y) {

console.log("x равно y");

} else {

console.log("y больше x");

}

if (z > y && z < x) {

console.log("z находится между x и y");

} else if (z === y || z === x) {

console.log("z равно x или y");

} else {

console.log("z находится за пределами x и y");

}

Какой будет вывод в консоль при следующих значениях переменных?

"x больше y" и "z находится за пределами x и y".

Задание 1.5

Какие значения будут выведены в консоль?

// Вариант A:

for (let i = 0; i ‹ 5; i++) {

console.log(i);

}

// Вариант B:

let j = 0;

while (j ‹ 5) {

console.log(j);
j++;

}

// Вариант C:

let k = 0;

do {

console.log(k);
k++;

}

// Вариант D:

let arr = [1, 2, 3, 4, 5];

for (let elem of arr) {

console.log(elem);

}

Вариант A выведет в консоль:

0,1,2,3,4

Вариант B выведет в консоль:

0,1,2,3,4

Вариант C выведет в консоль:

0,1,2,3,4

Вариант D выведет в консоль:

0,1,2,3,4
