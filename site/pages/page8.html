8. Заключение

Функции — один из самых важных и мощных инструментов в JavaScript. Они позволяют создавать модульный и переиспользуемый код, улучшать читаемость и облегчать его тестирование.

JavaScript предоставляет множество способов объявления и использования функций. Например, функции-объекты, анонимные функции, стрелочные функции, функции высшего порядка и замыкания.

А знание областей видимости, контекста вызова и способов передачи параметров функции критически важно для создания и использования функций в JavaScript.

Понимание всех этих аспектов поможет вам создавать эффективный и масштабируемый код на JavaScript.

Задание 8.1

Напишите функцию, которая принимает массив чисел в качестве аргумента и возвращает наибольшее число в массиве.

Пример использования:

const numbers = [3, 5, 1, 8, 2];

const result = findMax(numbers);

console.log(result); // 8

Посмотреть код

const findMax = function (numbers) {

let max = numbers[0];
for (let i = 1; i ‹ numbers.length; i++) {
if (numbers[i] > max) {
max = numbers[i];
}
}
return max;

};

Задание 8.3

Напишите IIFE-функцию, которая возвращает новый массив, содержащий квадраты чисел из исходного массива.

Посмотреть код

const arr = [1, 2, 3, 4, 5];

(function (arr) {

return arr.map(function (num) {
return num * num;
});

})(arr);

Задание 8.4

Напишите IIFE-функцию, которая принимает объект со свойствами width и height, и возвращает площадь прямоугольника, вычисленную по формуле width * height.

Посмотреть код

const rectangle = { width: 5, height: 10 };

(function (rect) {

return rect.width * rect.height;

})(rectangle);
