2. Знакомство с функциями

Функции могут быть созданы разными способами. Но чаще всего для этого используют ключевое слово function.

Давайте создадим простую функцию и разберём её структуру.

function helloSkillFactory() {

// тут располагаются инструкции

}

Структура у всех функций при создании в JavaScript одинаковая. Она состоит из следующих частей:

  1. Ключевое слово function, которое говорит JavaScript о том, что далее будет описана функция;
  2. Имя функции, по которому можно эту функцию вызывать в разных частях программы. В примере выше это helloSkillFactory;
  3. Круглые скобки (), в которых можно передавать аргументы функции;
  4. Фигурные скобки {}, внутри которых располагаются инструкции, исполняющиеся каждый раз при вызове функции. Также всё, что находится внутри этих фигурных скобок, называют телом функции.

Остальные способы создания функций рассмотрим в следующем юните. А сейчас детально разберём этот способ.

В примере выше функция helloSkillFactory ничего не делает и не возвращает никаких результатов. Теперь создадим функцию, которая будет выполнять самую простую операцию — вывод информации в консоль.

function helloSkillFactory() {

console.log('Привет, студент SkillFactory!')

}

Круто! Теперь наша функция что-то умеет делать. Давайте научимся её вызывать.

Вызов функции происходит просто. Достаточно использовать имя, которое было дано функции при создании и круглые скобки (). Они и инициируют вызов функции.

function helloSkillFactory() {

console.log('Привет, студент SkillFactory!')

}

helloSkillFactory()

Так при вызове функции в консоль мы увидим текст, который передали в функцию console.log.

Рекомендуем самостоятельно запускать все примеры, которые приведены в этом и других модулях. Это позволит вам быстрее освоиться в коде и чувствовать себя увереннее.

В случае с функциями в JavaScript вы можете использовать как консоль любого браузера, так и специализированные инструменты. Например, программа RunJS позволяет быстро запускать функции и видеть их результат (или ошибки).

В функции helloSkillFactory мы используем самую простую конструкцию, которую только можно представить в JavaScript — вывод в консоль с помощью функции console.log. Но в функциях могут быть разные по сложности конструкции. Например, создание переменных, циклы, условные конструкции, вызов других функций и так далее.

Проще говоря — в теле функций может быть любой JavaScript-код. В следующих примерах мы постепенно будем повышать сложность используемого тела функции, чтобы вы могли увидеть большую часть их возможностей.

Теперь поговорим про аргументы.

Аргументы функции

Аргументы функции ещё называют параметрами функции. Это набор значений, которые передаются в функции при вызове. Эти значения могут быть использованы в теле функции для вычислений. Аргументы могут быть любого типа данных. Например, числа, строки, объекты и даже другие функции.

Аргументы функции указываются внутри круглых скобок () после имени функции, разделённые запятыми.

function showText(text) {

console.log(text)

}

function sum(a, b) {

const result = a + b;
console.log('Сумма: ', result)

}

function multiply(a, b, c) {

let result = a * b * c;
if (result ‹ 10) {
console.log('Произведение меньше 10')
} else {
console.log('Произведение больше 10')
}

}

showText('Этот текст будет показан в консоли, круто!')

sum(2, 2) // 4

multiply(1, 2, 3) // “Произведение меньше 10”

В примерах выше приведены несколько функций с разным количеством аргументов — от 1 до 3. Но JavaScript никак не ограничивает в их количестве. Поэтому у вас может быть 5, 10 или даже 20 аргументов.

Но лучше не использовать их слишком много. Такой код трудно поддерживать и читать другим специалистам. В среднем разработчики используют до 3 – 5 аргументов.

Обратите внимание, что в функциях sum и multiply мы используем не только вывод значений в консоль, но также создаём переменные result, где сохраняем результат вычисления. А в функции multiply мы используем условную конструкцию для того, чтобы понять, больше или меньше 10 получившееся произведение.

Поэкспериментируйте с разными аргументами в функциях sum и multiply и посмотрите, как будет меняться результат. Попробуйте изменить значения аргументов или не использовать их вовсе.

Функции также могут принимать аргументы по умолчанию. Такие аргументы будут использованы, если при вызове функции аргумент не передан. Вот пример функции, которая принимает два аргумента и устанавливает значение по умолчанию для второго аргумента:

function greet(name, greeting = "Привет") {

console.log(greeting + ' ' + name + '!');

}

greet("Джон Сноу"); // "Привет Джон Сноу!"

greet("Джон Сноу", "Привет"); // "Привет Джон Сноу!" в консоль

В этом примере функция greet принимает два аргумента: name и greeting. Если значение для greeting не передано при вызове функции, то используется значение по умолчанию "Привет".

Как видите, мы можем использовать один и тот же участок кода (читай — функцию) для получения разного результата в зависимости от входящих аргументов. В своё время концепция функций сильно продвинула языки программирования вперёд, сделала их более гибкими и функциональными.

Но это ещё не все возможности функций, которые существуют. Разберём ещё одну важную концепцию.

До этого момента все рассмотренные нами функции что-то делали, но не сохраняли результат их выполнения. Для этой цели нам понадобятся возвращаемые значения.

Возвращаемое значение функции

Возвращаемое значение функции — это значение, которое функция возвращает при её вызове. Это может быть любое значение, включая примитивные типы данных (например, числа, строки, логические значения), объекты и даже функции.

Чтобы вернуть значение из функции, мы используем ключевое слово return. Немного модифицируем функцию sum из прошлого примера.

function sum(a, b) {

const result = a + b
return result

}

let resultSum = sum(1, 2)

let resultAnotherSum = sum(3, 7)

console.log('Сумма: ', resultSum) // "Сумма: 3"

console.log('Другая сумма: ', resultAnotherSum) // "Другая сумма: 10"

Теперь функция sum не просто вычисляет сумму значений, переданных ей в аргументах. Она также возвращает результат, который можно использовать дальше по ходу выполнения программы.

Ещё один небольшой пример, но теперь с возвращаемым объектом.

function createUser(name, age) {

return {
userName: name,
userAge: age
}

}

let jediUser = createUser('Оби-ван Кеноби', 74)

console.log('Новый пользователь: ', jediUser) // "'Новый пользователь: ' { userName: 'Оби-ван Кеноби', userAge: 74 }"

Как упоминалось выше, функции могут возвращать и другие функции. Но об этом подробнее поговорим позже, в юните 8.4 «Использование функций».

На этом знакомство с функциями подходит к концу. В этом юните были перечислены все базовые концепции работы функции, актуальные не только в JavaScript, но и во многих других языках программирования. Например, в Python, Go, Java и так далее.

В следующем юните детально поговорим про остальные способы объявления функций в JavaScript.
